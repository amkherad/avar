# 📋 صف‌ها

صف‌ها دانلودها را سازمان‌دهی می‌کنند و زمان‌بندی را کنترل می‌کنند.

## 📌 نوار کناری (داشبورد)

نوار کناری چپ صف‌ها را در نمای **فشرده** نشان می‌دهد — فقط نام صف دیده می‌شود. با قرار دادن نشانگر روی صف، توضیحات، وضعیت اجرا/توقف و تعداد دانلود نمایش داده می‌شود.

روی یک صف کلیک کنید تا فهرست دانلود فیلتر شود. با **مدیریت** صفحه مدیریت صف‌ها در **تنظیمات** باز می‌شود.

## ⚙️ تنظیمات ← صف‌ها

صفحه صف‌ها در تنظیمات یک **جدول** کامل دارد:

- **جستجو** — فیلتر صف‌ها بر اساس نام، توضیحات، وضعیت یا تعداد دانلود
- **مرتب‌سازی ستون** — روی سرستون کلیک کنید؛ کلیک دوم جهت را برعکس می‌کند؛ کلیک سوم مرتب‌سازی را پاک می‌کند
- **تغییر اندازه ستون** — لبه سرستون را بکشید
- **فیلتر وضعیت** — از منوی کشویی در سرستون وضعیت استفاده کنید
- **انتخاب چندگانه** — چک‌باکس‌ها را فعال کنید و چند صف را برای عملیات دسته‌ای انتخاب کنید
- **منوی زمینه** — راست‌کلیک روی ردیف برای شروع، توقف، ویرایش یا حذف

### عملیات دسته‌ای

وقتی یک یا چند صف انتخاب شده باشد، نوار ابزار دکمه‌های **شروع**، **توقف** و **حذف** را نشان می‌دهد. حذف دسته‌ای قبل از حذف همه صف‌های انتخاب‌شده تأیید می‌خواهد.

## ▶️ عملیات صف

| عملیات | توضیح |
|--------|--------|
| **شروع** | پردازش دانلودهای صف را آغاز می‌کند |
| **توقف** | صف را متوقف می‌کند |
| **افزودن** | صف جدید می‌سازد |
| **ویرایش** | توضیحات صف را ویرایش می‌کند |
| **حذف** | صف را حذف می‌کند (دانلودها از صف جدا می‌شوند) |

صف **پیش‌فرض** مصنوعی است و دانلودهای بدون صف اختصاصی را نشان می‌دهد. نمی‌توان آن را شروع، متوقف، ویرایش یا حذف کرد.
