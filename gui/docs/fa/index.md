# رابط کاربری آور

به **آور** خوش آمدید — مدیر دانلود با رابط وب و دسکتاپ. این رابط از طریق HTTP به دیمن آوار متصل می‌شود.

## بخش‌های اصلی

| بخش | محل | کاربرد |
|-----|-----|--------|
| **سربرگ** | نوار بالا | نام برنامه، بازگشت، پوسته، راهنما و تنظیمات |
| **نوار کناری** | چپ (همه صفحات) | صف‌ها، موضوعات راهنما یا گروه‌های تنظیمات |
| **نشست** | پایین نوار کناری | انتخاب و مدیریت نشست دیمن |
| **دانلودها** | وسط | جستجو، افزودن، انتخاب و کنترل دانلودها |
| **پنل جزئیات** | راست (اختیاری) | اطلاعات و عملیات دانلود انتخاب‌شده |
| **پاصفحه** | پایین | وضعیت سرور، کنسول و پنل جزئیات |
| **کنسول** | کشوی پایین | گزارش‌های رابط و دیمن |

## پیمایش

- فلش **بازگشت** در سربرگ — بازگشت به داشبورد از صفحات دیگر.
- آیکون **تنظیمات** — باز کردن تنظیمات.
- آیکون **راهنما** — مستندات داخلی.
- **Ctrl+1** — بازگشت به داشبورد.

موضوعات دیگر را در فهرست کناری راهنما ببینید.
