# رابط کاربری آوار

به **آوار** خوش آمدید — مدیر دانلود با رابط وب و دسکتاپ. این رابط از طریق HTTP به دیمن آوار متصل می‌شود.

## بخش‌های اصلی

| بخش | محل | کاربرد |
|-----|-----|--------|
| **سربرگ** | نوار بالا | نام برنامه، پوسته، راهنما و تنظیمات |
| **نوار کناری** | چپ (صفحه اصلی) | صف‌ها و نشست دیمن |
| **دانلودها** | وسط | جستجو، افزودن، انتخاب و کنترل دانلودها |
| **پنل جزئیات** | راست (اختیاری) | اطلاعات و عملیات دانلود انتخاب‌شده |
| **پاصفحه** | پایین | وضعیت سرور، کنسول و پنل جزئیات |
| **کنسول** | کشوی پایین | گزارش‌های رابط و دیمن |

## پیمایش

- آیکون **تنظیمات** — باز کردن تنظیمات (جایگزین کل فضای کار).
- آیکون **راهنما** — مستندات داخلی.
- **Ctrl+1** — بازگشت به داشبورد.

موضوعات دیگر را در فهرست کناری راهنما ببینید.
