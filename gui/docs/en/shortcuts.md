# Keyboard shortcuts

Shortcuts work globally except when focus is in a text field.

## Navigation

| Action | Default |
|--------|---------|
| Main dashboard | Ctrl+1 |
| Settings | Ctrl+, |
| Help | F1 |

## Downloads

| Action | Default |
|--------|---------|
| Add download | Ctrl+N |
| Focus search | Ctrl+F |
| Pause / resume selection | Ctrl+P |
| Start selection | Ctrl+Shift+S |
| Stop selection | Ctrl+Shift+X |
| Delete selection | Delete |

## View

| Action | Default |
|--------|---------|
| Toggle console | Ctrl+` |
| Toggle detail panel | Ctrl+D |

## Remapping

Go to **Settings → Shortcuts**. Click the key badge next to an action and press the desired combination.

Modifiers: **Ctrl** (or **Cmd** on macOS), **Shift**, **Alt**, plus a key.

Duplicate bindings are not prevented; the first matching action wins.
